Python Paypal PayflowPro Client
-------------------------------

This library provides an interface to make transactions with the PayPal PayflowPro API.

Requirements
------------
- Python 2.5+

The recurring payment functionality of this library requires the PayflowPro 
account to have the recurring payment functionality activated.


Usage
-----

Refer to the client.py file in the tests subdirectory for example usage of the client.

To run the tests you will have to edit the file and set the following variables from PayflowPro

PARTNER_ID
VENDOR_ID
USERNAME
PASSWORD
